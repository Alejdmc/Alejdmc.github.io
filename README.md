# Política de Privacidad – Carrera Arango S.A.S.

Este repositorio contiene la Política de Privacidad oficial de la aplicación móvil desarrollada para Carrera Arango S.A.S.

La política describe cómo se recopilan, utilizan y protegen los datos cuando se utiliza la aplicación de gestión de formularios vehiculares.

## Propósito

Este repositorio existe únicamente para publicar y mantener disponible la Política de Privacidad requerida para la distribución de la aplicación en App Store.

Apple exige que las aplicaciones que recopilan o procesan información de usuarios proporcionen una política de privacidad accesible públicamente mediante una URL.

## Alcance

La política aplica a la aplicación móvil utilizada para:

- Gestión de formularios vehiculares
- Registro de información técnica de vehículos
- Generación de documentos en PDF
- Envío de formularios a clientes o destinatarios internos

## Tecnologías relacionadas

La aplicación puede utilizar servicios externos para su funcionamiento, incluyendo:

- Supabase (base de datos, autenticación y almacenamiento)
- Servicios de correo electrónico para envío de documentos
- Integraciones mediante webhooks o herramientas de sincronización
